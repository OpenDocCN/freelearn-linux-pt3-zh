本书的这一部分包含以下章节：

+   *第十四章**，使用 journald*

+   *第十五章**，使用 systemd-networkd 和 systemd-resolved*

+   *第十六章**，理解 systemd 的时间管理*

+   *第十七章**，理解 systemd 和启动加载程序*

+   *第十八章**，理解 systemd-logind*
